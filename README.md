# FER - Programming Language Translation

Laboratory solutions for Programing Langugae Translation during the academic year 24/25.

Lab 1 (Java) - Implement a lexical analyzer for the simple programming language PJ\
Lab 2 (Java) - Implement a syntax analyzer for the programming language PJ\
Lab 3 (Java) - Implement a semantic analyzer for the programming language PJ

The PJ language is a made up languge for the laboratory exercises. It provides support for integer arithmetic and allows an arbitrary number of integer variables to store intermediate results.
